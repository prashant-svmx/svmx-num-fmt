Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @pcodebase/svmx-num-fmt`

## Usage

    var numFormatter = require('@pcodebase/svmx-num-fmt');

    var formattedNum = numFormatter(35666);


  Output should be `35,666`


## Tests
